# 简易计算器

## 功能说明
- 支持基本四则运算（加减乘除）
- 支持小数运算
- 支持键盘输入（数字键和回车键）
- 带错误提示功能
- 一键清除功能

## 环境要求
- Python 3.6+
- Tkinter库（通常随Python默认安装）

## 使用方法
1. 安装Python环境
2. 运行程序：
   ```bash
   python calculator.py
   ```

3. 界面操作：

   - 点击数字按钮或使用键盘输入数字
   - 使用运算符按钮进行计算
   - 按"="按钮或回车键计算结果
   - 按"C"按钮或ESC键清除输入
